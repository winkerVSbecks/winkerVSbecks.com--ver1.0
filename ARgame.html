AR Video Game

(Student Work)

My first project working within the processing environment. It's a very basic attempt at creating an augmented reality video game without actually using an AR tag decoding library. ARtoolkit (in processing) tends to be quite resource intensive and was unavilable for OSX at that time, so I decided to get a similar result by using blob detection and placing shapes on a white wall.

The game itself is a simple shoot 'em up where the enemy appears at random intervals and shoots at you. The user has to aim for these aliens by moving around the macbook and using the built in accelerometer to move the gun along the x-axis.

I was only responsible for the coding, all graphic elements and the sprites had to be used from online sources due to time constraints.

Made with Processing.
